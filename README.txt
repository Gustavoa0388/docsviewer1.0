# DocView Ortobio

## Descrição
Este é um aplicativo para gerenciar e visualizar documentos da Ortobio.

## Requisitos
- .NET Framework 4.8 ou superior
- Windows 10 ou superior

## Instalação
2. Execute o instalador Setup DocsViewer Ortobio para instalar o aplicativo.

## Como Usar
1. Abra o aplicativo DocsViewer Ortobio.exe.
2. Faça login com seu usuário e senha.
3. Utilize as funcionalidades do aplicativo conforme necessário.
